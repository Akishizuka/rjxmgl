# 项目启动说明

> 适用于第一次运行本项目，或需要向其他同学介绍如何启动时使用。

---

## 一、环境要求

| 项目 | 版本要求 |
|------|----------|
| Node.js | 18.x（本项目已适配，无需升级） |
| Python | 3.9+ |

> 注：项目原始配置要求 Node.js 20+，但我们已降级相关依赖以兼容 Node.js 18，**直接使用即可**。

---

## 二、项目结构

```
rjxmgl-master/
├── backend/          # 后端（FastAPI + SQLite）
│   ├── start.bat     # 后端一键启动脚本
│   ├── venv/         # Python 虚拟环境（已配置好）
│   └── xmgl.db       # 数据库文件
├── frontend/         # 前端（Vue 3 + Vite）
│   ├── package.json
│   └── vite.config.ts
└── 启动说明.md       # 本文件
```

---

## 三、启动步骤

### 1. 启动后端

打开 **终端 1**，执行：

```powershell
cd backend
.\start.bat
```

启动成功后显示：

```
Uvicorn running on http://0.0.0.0:8000
```

**保持此窗口运行**，不要关闭。

---

### 2. 启动前端

打开 **终端 2**（重新开一个），执行：

```powershell
cd frontend
npm install      # 首次运行需要安装依赖
npm run dev
```

启动成功后显示：

```
VITE v5.4.21  ready in xxx ms

➜  Local:   http://localhost:5173/
```

**保持此窗口运行**，不要关闭。

---

### 3. 访问项目

打开浏览器，访问：

```
http://localhost:5173
```

## 四、登录与注册

### 没有默认账号
本项目**没有预设的默认账号密码**，需要自行注册。

### 注册方法
1. 在登录页点击 **"注册"** 按钮
2. 填写账号和密码：
   - 账号：至少 3 位
   - 密码：至少 4 位
3. 点击注册，成功后自动跳转登录

### 已有账号
数据库中已有一个测试账号（密码未知）：
- 账号：`testuser5`

建议直接注册新账号使用。

## 五、常见问题

### Q1：前端报错 `crypto.hash is not a function`
**原因**：Vite 8 要求 Node.js 20+

**解决**：本项目已降级 Vite 至 5.x，如仍遇到此问题，请确认是否使用了本项目的 `package.json` 和 `vite.config.ts` 文件。

---

### Q2：注册/登录提示失败，控制台显示 `ECONNREFUSED ::1:8000`
**原因**：Node.js 18 将 `localhost` 解析为 IPv6，但后端监听在 IPv4

**解决**：本项目已将 `vite.config.ts` 中的代理地址从 `localhost:8000` 改为 `127.0.0.1:8000`，如仍遇到，请确认使用的是本项目的配置文件。

---

### Q3：`npm install` 安装很慢或卡住
**解决**：换用国内镜像源

```powershell
npm config set registry https://registry.npmmirror.com
```

---

### Q4：如何停止服务？
在两个终端中分别按 `Ctrl + C` 即可停止。
